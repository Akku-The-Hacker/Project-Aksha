# 📘 Aksha — Daily Routine Tracker

> Track the gap between what you plan and what you do. Build real discipline.

**Aksha** is a private, device-first daily routine tracking app for Android. It helps you compare planned time vs actual time, reflect honestly on your behavior, and build long-term discipline — without cloud dependency, ads, or dark patterns.

---

## ✨ Features

| Feature                    | Description |
|---------                   |-------------|
| 📅 **Smart Scheduling**    | Plan daily routines with precise time blocks, repeat days, and overnight support |
| ⏱️ **Planned vs Actual**   | Track the real gap between intention and execution |
| 📊 **Rich Insights**       | Daily, weekly, and monthly analytics with performance & consistency metrics |
| 🎮 **Gamification**        | Earn achievements (Week Warrior, Perfect Day, Early Bird...), build streaks |
| 🏷️ **Categories & Tags**   | Organize routines with color-coded categories and custom tags |
| 📝 **Daily Reflections**   | End-of-day journaling to build self-awareness |
| 🔔 **Smart Notifications** | Per-routine reminders with customizable timing |
| 💾 **Backup & Export**     | Google Drive backup + Excel export for full data portability |
| 📆 **Calendar View**       | Visual overview of your routine history |
| 🌙 **Dark Mode**           | Beautiful light & dark themes |

---

## 🔒 Privacy First

- **100% offline** — works without internet
- **Device-first** — all data stored on your phone
- **No ads, no trackers, no analytics**
- **No subscription fees**
- **You own your data completely**

> *"Aksha does not collect, transmit, or store your personal data on any server. All your data stays on your device."*

## 📱 Download

👉 **[Download Latest APK](https://github.com/Akku-The-Hacker/aksha/releases/latest)**

**Requirements:** Android 6.0+ (API 23)

### Installation
1. Download the APK from the link above
2. Open the file on your Android device
3. If prompted, enable "Install from unknown sources"
4. Install and start tracking!
